## How to Use This Dashboard

1. **Start the Watcher**: Run `python watchers/file_watcher.py` to begin monitoring for triggers
2. **Start Dashboard Updater**: Run `python watchers/dashboard_updater.py` to refresh this dashboard every 30 seconds
3. **Monitor Progress**: Task counts update automatically as files move through the system
4. **Check Recent Activity**: Completed tasks appear here with timestamps

**Note**: If dashboard doesn't auto-update, Obsidian may have a file lock. Close the file in Obsidian or refresh the window.
